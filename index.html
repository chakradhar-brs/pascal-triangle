Pascal's Triangle

The displayTriangle function first gets the value of a text input field with the id "rows" and converts it to an integer. If the value is not a positive integer, an error message is displayed and the function returns. Otherwise, the function generates the Pascal's triangle using the generatePascalsTriangle function and an empty HTML element with the id "triangle". It then creates a new div element for each row of the triangle, and for each element in the row, it creates a new div element, sets its text content to the value of the element, and appends it to the row div.
